# XRP Robot User's Guide

*for Cornell XRP System Proving Ground Competition* *By INCOSE 2025 Student Group — Version 3, Updated Spring 2026*

------------------------------------------------------------------------

Welcome to the Systems Engineering MEng Program! This guide, developed by the INCOSE Cornell Student Division, will help you understand the XRP robot and complete the Inaugural Challenge's RFP. It does not provide challenge solutions — it gives you the basic knowledge to build your own robotic solutions.
